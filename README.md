# 📄 Contract

Sistema de gestão de contratos desenvolvido em **PHP**, **HTML**, **Bootstrap**, **jQuery** e **MySQL**, com geração automática de documentos em PDF utilizando a biblioteca **FPDF**.

O objetivo do projeto é centralizar o gerenciamento de clientes, serviços e contratos, proporcionando um ambiente simples e eficiente para controle de documentos contratuais.

---

## ✨ Funcionalidades

### 👥 Gestão de Clientes

- Cadastro de clientes;
- Edição de informações;
- Exclusão de registros;
- Consulta e listagem de clientes.

### 🛠️ Gestão de Serviços

- Cadastro de serviços;
- Atualização de informações;
- Exclusão de serviços;
- Organização e controle dos serviços oferecidos.

### 📑 Gestão de Contratos

- Associação de clientes e serviços aos contratos;
- Armazenamento de contratos em banco de dados;
- Consulta e gerenciamento de contratos cadastrados;

### 📄 Geração de PDF

- Emissão automática de contratos;
- Geração de documentos em formato PDF através da biblioteca FPDF;
- Layout padronizado para impressão e compartilhamento.
